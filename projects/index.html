Projects

During my undergraduate study, I have participated in several projects in robotics, controls, computer vision, graphics, and modelling. I am really looking forward to conduct any relative research in my graduate study. These past endeavors helps me build up my abilities to explore the core principles of topics and solve practical problems independently. Besides, I also found that I enjoyed working on a multi-disciplinary team because our diverse fields helped us accomplish our goals effectively and efficiently.

Robotics & Controls

  • Navigation and Human Robot Interaction on the Segway RMP Platform
    Advisor: Prof. Martin Jagersand the University of Alberta, Edmonton, AB, Canada Jul. 2014 - Oct. 2014
    I participated in UARE Summer Intership in the Robotics Lab at the University of Alberta. My task was to interact with the Segway robot with gestures using a Microsoft Kinect RGB-D camera. I intergrated the ROS Navigation Stack to the Segway RMP platform, and adatped the existing user interface acheived by OpenNI and Point Cloud Library (PCL).

  • Trials for International Design Contest (IDC) Robocon in Zhejiang University
    Advisor: Prof. Rong Xiong Zhejiang University, Hangzhou, China Mar. 2014 - Apr. 2014
    In the competition, each team was required to design and implement robots with limited materials. The goal was to place cubes on the "mountain". The higher the cubes were placed the higher the score was earned. There were some additional tasks to earn bonus, e.g., shooting tennis balls and drawing a damped drawbar. Finally, We won the championship!

  • Soccer Game Project on NAO Humanoid Robot Platform
    Advisor: Prof. Wei Jiang & Dr. Chunlin Zhou Zhejiang University, Hangzhou, China Sept. 2013 - Jan. 2014
    The task was to make the biped robot walk steadily and kick the ball. A linkage model in Matlab was built for the humanoid robot in order to solve kinematics problems, in which the matrix method proposed by Denavit and Harterberg was applied to describe the translational and rotational relationship between joints. Meanwhile, gaits for the humanoid robot were planned to keep the zero moment point (ZMP) within its support polygon.

  • Path Planning and Obstacle Avoidance on NI LabVIEW Robotics Starter Kit (DaNI)
    Advisor: Prof. Rong Xiong & Dr. Chunlin Zhou Zhejiang University, Hangzhou, China Mar. 2013 - Jun. 2014
    In the project, I made the DaNI robot traveled through a maze successfully in accordance with a given grid roadmap. Localization was achieved by dead reckoning, while global path planning and local obstacle avoidance were implemented with A* search algorithms and vector field histogram (VFH) respectively.

Vision & Graphics

  • Triangulation and Surface Reconstruction from Unordered Point Clouds
    Advisor: Prof. Wei Jiang Zhejiang University, Hangzhou, China Jun. 2014 - Present
    To better display the result of the project described below, I started to work on triangulation and surface reconstruction. Principal component analysis (PCA) and 2D Delaunay triangulation was used at first, but the result was not desirable. Greedy Surface Triangulation algorithm was then applied for a faster speed and a better accuracy. Currently, I am still searching for a better solution.

  • Depth Recovery from Monocular Video Sequences
    Advisor: Prof. Wei Jiang Zhejiang University, Hangzhou, China Mar. 2014 - Present
    The idea of depth recovery from monocular video sequences stemed from template matching for object recognition. Scale-Invariant Feature Transform (SIFT) and Random Sample Consensus (RANSAC) were applied to synthesizes 3D feature point clouds from selected frames. Currently, I am working on improving the efficiency and robustness by adopting more compact descriptors and more advanced multi-model feature matching algorithms such as Adaptive Multi-Affine (AMA) and Hierarchical Multi-Affine (HMA).

  • Object Recognition for Home Environment Service Robot
    Institute of Cyber-Systems & Control Zhejiang University, Hangzhou, China Oct. 2013 - Jul. 2014
    Object recognition is a module of great importance for home environment service robots. Since only by object recognition, can the robot know whether it should bring the Cola or the Sprite. Spending a year implementing and testing various feature detectors and descriptors in the laboratory, I built an object recognition module using Speeded Up Robust Features (SURF) under both LINUX and Robot Operating System (ROS).

  • Ping-Pong Ball Detection with OpenCV
    Advisor: Prof. Wei Jiang Zhejiang University, Hangzhou, China Sept. 2013 - Jan. 2014
    The Ping-Pong ball was detected with circle Hough Transformation and various filters. I also applied mathematical morphology to remove noises. However, due to shadows and variations of lighting conditions, frequent changes of the thresholds were required. After scrutinizing materials on artificial intelligence, I trained a neural network classifier to evaluate pixels inside potential circles and worked out a system of high recognition rates and reliability.

  • Android Application Based on 3D Point Clouds and Mesh Grids Display
    Advisor: Prof. Rong Xiong Zhejiang University, Hangzhou, China Mar. 2013 - Apr. 2014
    It was my first Android Application. It achieved communication with computers and display of 3D point clouds and mesh grids. The App was originally designed for 3D mesh map display, but it usage was limited due to the poor performance of the android phone. With only some knowledge about programming in C, I taught myself java and android programing in few weeks, and implemented the core module of display with the OpenGLES library.

Modelling

  • Recycling Ethanol from Given Effluent
    Advisor: Prof. Mei Dou Zhejiang University, Hangzhou, China Dec. 2013
    In order to simulate the whole process and determined the optimum feed location and reflux ration of a packed tower model, We built a model with Matlab. The model result was contracted with the result of Aspen simulation. In the project, I mainly worked on mechanism modeling with Matlab.

  • Analysis on the Main Factors for Waste Reduction and Classification
    Advisor: Prof. Kangsheng Liu Zhejiang University, Hangzhou, China May. 2013
    Our mission was to analyze the main factors for the performance of waste reduction and classification. My team built the model with analytic hierarchy process (AHP), and proved the model with regression analysis relying on the statistics data of each waste category. Feasible suggestions were offered to the government and prediction for the situation of the next five year was made based on a grey box model.
